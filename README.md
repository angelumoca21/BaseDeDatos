# Base de datos

### *Duración: 30 horas (15 sesiones [Lunes a jueves] y 7 sabatinas)*

## Objetivo general

El estudiantado aplicará conocimientos básicos de la estructura de base de datos en lenguaje de consulta SQL mediante ejercicios prácticos y reales, con el propósito de guardar, organizar y almacenar datos para fácil manejo.

## Evaluación

- 50% Actividades en sesiones
- 50% Proyecto final
